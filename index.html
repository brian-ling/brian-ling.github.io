Course Description:

Introduction to R, data creation and manipulation, data import and export, scripts and functions, control flow, debugging and profiling, data visualization, statistical inference, Monte Carlo methods, decision trees, support vector machines, neural network, numerical methods.

Intended Student Learning Outcomes:

  1. Understand the fundamental concepts in R
  2. Be able to import and tidy data for further analysis
  3. Be able to visualize data and perform exploratory data analysis
  4. Be able to apply appropriate methods for statistical analysis and interpret the output
  5. Be able to apply common machine learning algorithms with real-world applications
  6. Understand the basics of numerical and Monte Carlo methods

Course Outline:

  1. R Basic
  • Installing R, Rstudio and packages
  • Basic data types and operations
  • Data structures: vector, matrix, array, list, dara frame
  • Data input and output
  • Scripts and functions
  • Control flow
  1. Data visualization with ggplot2
  2. Data manipulation with tidyverse
  3. Statistical inference in R: one/two-sample test of mean and proportion; correlation test; non-parametric test; regression analysis
  4. Some machine learning methods: nearest neighbors, naive Bayes, decision trees, support vector machines, neural network, k-means clustering
  5. Introduction to numerical methods: Monte Carlo simulation, gradient descent, Newton Raphson algorithm

Note: actual topics may change slightly depending on our progress

Grading Scheme:

Assignments (30%):

Late submission: receive 80% of the points (within one day), receive 0 (after one day). No exception other than academic accommodation and academic consideration for an extenuating circumstance. Quizzes (30%):

Weekly quizzes. Simple questions will be posted on Thursday each week and each quiz is due on Sunday noon (receive 80% afterwards). You should be able to answer them after attending the lectures. One quiz with the lowest score will be dropped.
