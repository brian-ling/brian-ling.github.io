Syllabus

Course Description: Introduction to R, data creation and manipulation, data import and export, scripts and functions, control flow, debugging and profiling, data visualization, statistical inference, Monte Carlo methods, decision trees, support vector machines, neural network, numerical methods.

Teacher assistant: TBD

Office Hours:

Intended Student Learning Outcomes:

  1. Understand the fundamental concepts in R
  2. Be able to import and tidy data for further analysis.

Prerequisite

Textbooks/Readings

  1. R Markdown: The Definitive Guide
  2. R for Data Science, by Garrett Grolemund and Hadley Wickham

Course Outline

  1. Installing R, Rstudio and packages
  2. Basic data types and operations
  3. Data structures: vector, matrix, array, list, dara frame
  4. Data input and output
  5. Scripts and functinos
  6. Control flow
  7. Debugging and profiling
  8. Data visualization with ggplot2
  9. Data manipulation with tidyverse
  10. Statistical inference in R: one/two-sample test of mean and proportion; correlation test; non-parametric test; regression analysis
  11. Introduction to Monte Carlo methods
  12. Decision trees, support vector machines, neural network
  13. Introduction to numericacl methods: gradient descent, Newton Raphson algorithm
